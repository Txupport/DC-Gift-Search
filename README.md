Simply download the .exe in the releases section to any location on your PC and launch it.
You can type in a (Full or Partial) name of any monster or item in the search box at the top, press enter to search the monster index, or click what index you wish to search (Search For Monster / Search For Item).
If you wish to see exactly what the .exe is made of, the .pyw file is in the source code .zip.
